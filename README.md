This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).

# How to start
- initialize
```
yarn install
```
- develop:
```
yarn start
```
- build:
```
yarn build
```
- deploy to [github-page](https://zetsin.github.io/mobile-guard/)
```
yarn deploy
```
